<i><b>MyContactTracing</b></i> - Pseudo-contact-tracing-app developed for Honours Project (2020/2021).

<i>How to run:</i>
- Clone git repository
- Open the project in Android Studio
- Run on AVD manager (emulator) or physical Android Device
